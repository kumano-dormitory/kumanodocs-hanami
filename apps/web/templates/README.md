# 一般向けページのテンプレートファイル

このフォルダ内には一般向けページの全てのアクションに対応するテンプレートファイルがある

それぞれのファイルはERB形式で、変換されることでHTML形式になる

## レイアウトテンプレートファイル
一般向けページのレイアウトを決定するテンプレートファイルは`application.html.erb`である

特殊なレイアウトを利用するアクションは以下の通りである. その他のアクションは全て上記のレイアウトテンプレートを用いる
- 資料システムへのログインページ(`login.html.erb`)
- 議案の差分表示ページ(`diff.html.erb`)
- "このサイトについて"ページ(`about.html.erb`)

## エラーページ
HTTP STATUS CODEが400の場合のエラーページは`400.html.erb`である

同様にして、401, 403, 404, 422, 500の場合のエラーページもある
